## 🆘 **문제 해결**

### **가장 흔한 문제 5가지**

#### **1. 프로그램이 안 열려요**

```bash
# 확인 1: Python 설치 확인
python --version

# 확인 2: 파일 위치 확인
# run_tradingview_gui.bat이
# C:\Users\palet\Desktop\Factory\ib-trading-bot\ 에 있나?

# 확인 3: 수동 실행
python scripts/tradingview_gui_ultra_simple.py
```

#### **2. 종목 분석이 안 돼요**

```
❌ 잘못한 것:
- 입력하지 않고 버튼만 클림
- "📖 종목 분석" 버튼을 안 클림

✅ 올바른 방법:
1. 종목 입력: BTCUSDT, ETHUSDT, ...
2. "📖 종목 분석" 클릭
3. 리스트에 종목이 표시됨
```

#### **3. 클립보드 복사가 안 돼요**

```
❌ 잘못한 것:
- 입력만 하고 생성을 안 함
- "🚀 알림 생성" 실행 안 함

✅ 올바른 방법:
1. "📖 종목 분석" 클릭
2. "✓ 모두 선택" 클릭
3. "🚀 알림 생성" 클릭
4. "📋 모두 복사" 클릭
```

#### **4. 파일이 저장 안 돼요**

```
❌ 잘못한 것:
- "🚀 알림 생성"을 안 함

✅ 올바른 방법:
1. 먼저 알림 생성
2. "💾 파일 저장" 클릭
3. 파일 위치 선택
4. 저장
```

#### **5. Webhook 설정이 안 맞아요**

```
❌ 잘못한 것:
- .env와 프로그램 값이 다름

✅ 올바른 방법:
1. .env 파일 확인
   cat .env | grep WEBHOOK_SECRET
2. 프로그램의 설정 탭에서 같은 값 입력
3. "🔍 확인" 클릭
```

## 💬 **FAQ - 자주 묻는 질문**

### **Q: 외부 프로그램 설치해야 하나요?**

```
A: 아니요! Python만 있으면 됩니다.
   PyQt5, PySimpleGUI 등 설치 불필요.
   Tkinter는 Python 내장입니다.
```

### **Q: 인터넷 연결이 필요한가요?**

```
A: 프로그램 자체는 필요 없습니다.
   TradingView에 제출할 때만 필요합니다.
```

### **Q: 바이러스는 아닌가요?**

```
A: 100% 안전합니다.
   - 오픈소스 (누구나 코드 확인 가능)
   - Tkinter만 사용 (표준 라이브러리)
   - 백신 프로그램에서 허용
```

### **Q: 실제 거래를 시작하려면?**

```
A: 이 프로그램은 "JSON 생성기"입니다.
   실제 거래는 메인 봇이 처리합니다:

   TradingView → (알림 전송)
   → 웹훅 수신 → (main.py)
   → 큐 저장 → (Redis)
   → 워커 처리 → (order_worker.py)
   → IB 실행 → (ib_client.py)
   → 거래 완료!
```

### **Q: 같은 종목으로 여러 번 설정할 수 있나요?**

```
A: 네, 가능합니다.
   각 종목마다 다른 조건으로 설정 가능:
   - 다른 RSI 레벨
   - 다른 시간 프레임
   - 다른 금액

   프로그램은 JSON만 생성하므로
   TradingView에서 조건을 다르게 설정하면 됩니다.
```

### **Q: 오프라인에서 쓸 수 있나요?**

```
A: 네, 프로그램 자체는 완전 오프라인입니다.
   JSON을 생성하고 파일로 저장한 후
   나중에 TradingView에 입력할 수 있습니다.
```
