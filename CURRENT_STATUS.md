# 자동투자 시스템 현재 상태

최종 정리일: 2026-05-25

## 기준

- 이 폴더는 로컬 작업 복사본입니다.
- 실전 자동투자 실행 기준은 VPS의 `/root/ib-trading-bot`입니다.
- TradingView 웹훅 URL은 `http://1.234.65.43/webhook` 기준입니다.
- 민감정보는 문서에 저장하지 않습니다. 실제 값은 `.env` 또는 별도 비밀 저장소에서만 관리합니다.

## 2026-05-25 조치

- TradingView 웹훅 타임아웃 완화를 위해 웹훅 응답을 빠르게 반환하도록 개선했습니다.
- Redis 기반 idempotency guard를 추가해 같은 알러트 중복 처리를 막도록 했습니다.
- 개선된 파일은 VPS에 선별 배포했고, 배포 전 원격 백업을 남겼습니다.
- 이 로컬 폴더의 문서에 남아 있던 토큰, 웹훅 시크릿, 비밀번호 형태의 값은 마스킹했습니다.

## 주의

- 이 Mac에서 `8000`, `5432`, `6379` 포트가 떠 있지 않은 것은 곧바로 실전 장애를 뜻하지 않습니다.
- 현재 실전 실행 여부는 로컬 포트가 아니라 VPS의 Docker 컨테이너와 `/health` 응답으로 판단해야 합니다.
- 원본 폴더와 복사본 폴더가 둘 다 있어도, 웹훅 URL이나 실행 프로세스에 연결하지 않으면 실전 주문에는 영향을 주지 않습니다.
- 같은 TradingView 알러트를 수동 재전송하면 중복 주문 위험이 있으므로 먼저 VPS 로그와 Redis 큐를 확인해야 합니다.

## 확인 기준

```bash
curl -fsS http://1.234.65.43/health
ssh root@1.234.65.43 'cd /root/ib-trading-bot && docker compose ps'
```

비밀번호, 토큰, 계좌번호, 웹훅 시크릿은 이 문서에 적지 않습니다.
