## 🎯 **3단계 사용법**

### **Step 1️⃣: 종목 입력**

텍스트 박스에 입력:

```
BTCUSDT, ETHUSDT, AAPL, MSFT, GOOGL

또는 한 줄씩:
BTCUSDT
ETHUSDT
AAPL
```

### **Step 2️⃣: 버튼 클릭**

순서대로:

```
1. "📖 종목 분석" 클릭
   → 5개 종목 인식됨

2. "✓ 모두 선택" 클릭
   → 모든 종목 선택됨

3. "🚀 알림 생성" 클릭
   → 10개 알림 생성됨 (5 종목 × BUY/SELL)
```

### **Step 3️⃣: 결과 확인**

"📋 결과" 탭에서:

```
생성된 JSON이 모두 표시됨

{
  "secret": "MySecret123456",
  "action": "BUY",
  "ticker": "BTCUSDT",
  "price": "{{close}}",
  "time": "{{timenow}}"
}
```

---

## 📋 **Tab 설명**

### **Tab 1️⃣: 📊 와치리스트**

```
입력 → 분석 → 선택 → 생성
```

**기능:**
- 종목 자동 분석
- 모두 선택/해제
- 일괄 알림 생성

---

### **Tab 2️⃣: ✏️ 수동 추가**

```
종목 입력 → 동작 선택 → 미리보기 → 복사
```

**기능:**
- 단일 종목 설정
- BUY/SELL/둘다 선택
- JSON 미리보기
- 클립보드 복사

---

### **Tab 3️⃣: 📋 결과**

```
생성된 모든 JSON 표시
```

**기능:**
- 전체 결과 보기
- 모두 복사
- 파일 저장

---

### **Tab 4️⃣: ⚙️ 설정**

```
Secret/URL 확인 및 검증
```

**기능:**
- Webhook Secret 설정
- Webhook URL 설정
- 설정 검증

---

## 🧪 **실제 사용 예제**

### **5개 종목 설정하기**

```
1. 프로그램 실행
   run_tradingview_gui.bat

2. 종목 입력:
   BTCUSDT, ETHUSDT, AAPL, MSFT, GOOGL

3. "📖 종목 분석" 클릭
   → "✅ 5개 종목 인식됨"

4. "✓ 모두 선택" 클릭
   → 모든 종목이 파란색으로 표시

5. "🚀 알림 생성" 클릭
   → "✅ 10개 알림 생성 완료!"

6. "📋 결과" 탭에서 JSON 확인
   → 10개의 JSON 메시지 표시

7. TradingView에 각 JSON 복붙
   → 각 종목마다 BUY/SELL 알림 추가

8. 완료! ✅
```

---

## 📝 **TradingView에서 알림 추가**

### **각 JSON마다:**

1. **TradingView 차트 열기**
   ```
   https://www.tradingview.com/chart/?symbol=BTCUSDT
   ```

2. **벨 아이콘 → "알림 추가"**

3. **기본 설정**
   ```
   심볼: BTCUSDT
   조건: 당신의 조건 (RSI < 30 등)
   인터벌: 1일
   ```

4. **"메시지" 탭**
   - 생성된 JSON 복붙
   - "📋 복사" 버튼 사용하면 쉬움

5. **"알림" 탭**
   ```
   웹훅 URL: http://localhost:8000/webhook
   ☑️ 웹훅 URL 체크
   ```

6. **"생성" 클릭**

## 💡 **팁**

### **클립보드 복사가 안 되면:**

```
텍스트를 수동으로 선택
Ctrl+C로 복사
TradingView에 Ctrl+V로 붙여넣기
```

### **파일로 저장하려면:**

```
"📋 결과" 탭
"💾 파일 저장" 버튼 클릭
원하는 위치에 JSON 파일 저장
```

### **Webhook Secret 확인:**

```
"⚙️ 설정" 탭
"🔍 설정 검증" 버튼 클릭
```
